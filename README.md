
# Hacker News Feed app

### Prerequisities

In order to run this container you'll need docker installed.

* Docker v19.03.13
* Docker-compose v1.27.4

### Stack

- Node 14.15
- React 17
- NestJs - Typescript
- Docker

## Usage

On root dir 
`` $ docker-compose up --build  ``

Then you'll see the app running on 

``localhost:3000``
